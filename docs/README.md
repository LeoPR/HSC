# Documentação do HSC

| Eixo | Propósito | Pasta |
|------|-----------|-------|

| Framework | Visão teórica do HSC — o que este projeto propõe | [framework/](framework/) |

## Como ler

- **Começando do zero:** [framework/vision.md](framework/vision.md) dá o enquadramento conceitual.
- **Contexto técnico:** [literature/](literature/) cobre curvas espaciais, transformadas de frequência, reordenação simbólica, codificação entrópica, compressão fractal, transformações de feature space e compressão topológica.
- **Rastreamento científico:** [methodology/roadmap.md](methodology/roadmap.md) traz o plano de experimentos, e [methodology/siac.md](methodology/siac.md) descreve o método SIAC de condução.
- **Perguntas abertas:** ver [`../evaluations/`](../evaluations/).
